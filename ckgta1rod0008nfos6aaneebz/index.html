工欲善其事,必先利其器

发表于 2020-10-14 更新于 2020-11-12 分类于 工具配置 阅读次数：

    俗话说的好，工欲善其事必先利其器，作为一名有追求的开发者，当然要将自己吃饭的家伙打造的即好用又美观啦。

    接下来我主要会从系统软件与配置,Chrome插件,Terminal 三个方面去推荐好用的软件及配置

  • 系统软件与配置篇

    • 系统软件
      name description
      Alfred4 提升效率的神器不过多解释了，必装
      超级右键 可以增强mac的右键功能，在appstore就可以下载
      Dozer 免费开源的隐藏图标工具
      iterm2 终端，开发人员必备
      iStat Menus 状态栏对于系统资源的监控，很好用且美观
      Foldery 可以自定义文件夹的颜色的小工具
      OpenInTerminial 在Finder中可以直接打开终端，编辑器
      Snipaste 截图贴图的神器，不解释
      NeatDownloadManager 替代浏览器自身的下载器，可以最大开启32线程同时下载，很好用
      Tinycal 一个很简洁的状态栏日历
      Awesome-mac 其他mac软件
    • 系统设置
      • 通过如下命令，可以让那些包含隐藏窗口的应用程序图标变暗，从而方便我们区分
        defaults write com.apple.dock showhidden -bool true; killall Dock
      • 恢复为默认设置
        defaults delete com.apple.Dock showhidden; Killall Dock
      • 开启hidpi(当外接2k 4k显示器使用时，可以调整画面更细腻舒适)
        sh -c "$(curl -fsSL https://raw.githubusercontent.com/xzhih/one-key-hidpi/master/hidpi-zh.sh)"
      • 设置dock隐藏后 弹出响应时间
        defaults write com.apple.Dock autohide-delay -int 0 && killall Dock
        defaults write com.apple.dock autohide-time-modifier -int 0; killall Dock
      • 显示与恢复隐藏文件
        defaults write com.apple.finder AppleShowAllFiles -boolean true ; killall Finder
        defaults write com.apple.finder AppleShowAllFiles -boolean false ; killall Finder
      • 禁止与恢复.DS_store生成
        defaults write com.apple.desktopservices DSDontWriteNetworkStores -bool TRUE
        defaults delete com.apple.desktopservices DSDontWriteNetworkStores
    • other 身为开发人员必装的依赖command-line-tools

      • 命令行安装方法
        xcode-select --install

      • 检查是否安装成功
        xcode-select --print-path

      • 如果命令行安装失败，可以直接去苹果开发者网站下载安装

  • Chrome插件篇

    好用的插件很多，但是在经过我长时间的使用之后留下的只有下面的这些，这里也提供一个谷歌插件英雄榜，可以挑选适合自己的插件

    name description
    Adguard 全平台的广告拦截器，也可以只装插件，反正我是把用了许久的adblock换成这个了
    APK Downloader 用来下载google play中的软件
    File Icons for GitHub and GitLab 给github gitlab中的项目文件添加对应的图标
    NeatDownloadMnagaer NDM需要安装对应插件使用
    Octotree - GitHub code tree 显示github中的目录文件树，不用点来点去了，直接在侧边栏就可以看
    SimpleUndoClose 恢复手贱关掉的网页
    沙拉查词 很好用的翻译插件
    掘金 查看当下都有哪些流行的项目，和新项目
    Tampermonkey 不解释 懂的都懂

  • Terminal

    name description install usage
    brew 不解释必装 /bin/bash -c “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)" brew –help
    ohmyzsh 不解释必装 sh -c “$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)" /
    tldr 有啥命令不会用了就直接用它查看，还有使用示例 brew install tldr tldr wget
    mycli mysql增强客户端可以自动补全，记录历史，彩色 brew install mycli 同mysql-client
    lolcat 炫彩的cat brew install lolcat …
    httpie 功能同curl,但是要好用许多 brew install httpie http –help
    cmatrix 装逼 brew install cmatrix cmatrix
    fd 增强的find，比find要快很多 brew install fd fd –help
    procs 比ps更好用 brew install procs procs docker
    htop 众所周知的东西了 brew install htop htop
    neofetch 终端输出美化的电脑配置图 brew install neofetch neofetch
    telnet，tree mac 自带没有这些工具 brew install /
    the_silver_searcher 用来搜索文件中字符串的工具 brew install ag –help
    ncdu 交互式查看文件的工具 brew install ncdu
    loc 统计代码行数 brew install loc .
    fanyi 命令翻译工具 npm install fanyi -g fanyi xxx
