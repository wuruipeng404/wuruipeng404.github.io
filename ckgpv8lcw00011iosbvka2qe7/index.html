折腾之在CentOS8上安装K8s环境

发表于 2020-10-21 分类于 centos 阅读次数：

centos8 已经出来一段时间了，在刚出来的时候，尝试过使用一次，貌似相较7变化有些大，就没继续折腾
现在有时间了，来尝试一下把K8S的环境搭建在centos8上。

  1. 虚拟机

    我用的是Parallels Desktop,装的centos8虚拟机，用的镜像目前版本是 CentOS-8.2.2004-x86_64-minimal.iso

  2. IP

    跟之前不一样的是，8装好ifconfig是没有ip的，这就需要我们手动配置一下了。

    1. 查看 /etc/sysconfig/network-scripts/目录下并没有eth0网卡，不过有一个ifcfg-xxx，这里应该每个人是不一样的，不过不管他是什么我们都给改为，ifcfg-eth0.
    2. 同时还要修改ifcfg-eth0内的几个配置项 NAME和DEVICE 改为 eth0, ONBOOT 改为 yes.
    3. 编辑/etc/default/grub 在 GRUBCMDLINELINUX变量 的 crashkernel=auto 后面加上 net.ifnames=0 biosdevname=0
    4. 运行命令grub2-mkconfig -o /boot/grub2/grub.cfg来重新生成GRUB配置并更新内核参数.
    5. 最后重启，然后就可以看到ip啦，就可以直接ssh进行操作了。
    6. PS：centos8 已经没有newwork.service这个服务了，如果要单纯重启网络可以执行 nmcli c reload。
  3. 修改yum源为清华源
    1. 首先备份 sudo cp /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.bak
    2. 然后编辑 /etc/yum.repos.d/CentOS-Base.repo
      1. 注释 mirrorlist= 这一行
      2. 取消注释 baseurl= 这一行，并将 mirror.centos.org 替换为 mirrors.tuna.tsinghua.edu.cn
    3. 最后 执行 sudo yum makecache && sudo yum -y update
  4. 安装一些工具
    yum -y install python3 vim git wget telnet* the_silver_searcher
pip3 install mycli ipython httpie
    • htop
      原来htop安装 直接yum -y install 就可以的，现在需要用dnf
      1. dnf install https://dl.Fedoraproject.org/pub/epel/epel-release-latest-8.noarch.rpm
      2. dnf update
      3. dnf install htop
  5. 安装Docker
    1. 安装依赖 sudo yum install -y yum-utils device-mapper-persistent-data lvm2
    2. 配置阿里云仓库 sudo yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
    3. 由于centos8默认使用podman，所以我们要先手动安装containerd.io，同时如果你的源下载的版本太低，可以手动下载新版本 Download 进行安装
    4. yum -y install docker-ce
    5. 修改docker配置文件，镜像源加速 
      cat <<EOF > /etc/docker/daemon.json
{
    "registry-mirrors": ["https://registry.docker-cn.com","http://hub-mirror.c.163.com"],
}
EOF
    6. systemctl start docker
  6. 安装K8S

    1. 关闭防火墙

      systemctl disable firewalld
systemctl stop firewalld

    2. 禁用SELinux

      setenforce 0
# 或者
vim /etc/sysconfig/selinux
SELINUX=disabled
reboot

    3. 安装kubeadm(用于安装k8s集群的工具)

      # 修改源
vim /etc/yum.repos.d/kubernetes.repo
      [kubernetes]
name=Kubernetes Repository
baseurl=http://mirrors.aliyun.com/kubernetes/yum/repos/kubernetes-el7-x86_64/
enabled=1
gpgcheck=0
      yum -y install kubelet kubeadm kubectl --disableexcludes=kubernetes
      systemctl enable docker && systemctl start docker
systemctl enable kubelet && systemctl start kubelet

    4. kubeadm config

      # 初始化配置文件备用
kubeadm config print init-defaults > init.defaults.yaml
# 拷贝一份
cp init.defaults.yaml init.yaml

      修改 init.yaml 中的如下项

      localAPIEndpoint:
  advertiseAddress: 你的ip

imageRepository: docker.io/aiotceo # 镜像仓库地址

    5. 下载相关镜像

      kubeadm config images pull --config=init.yaml

    6. Master节点

      1. 关闭swap,修改/etc/sysconfig/kubelet的配置项KUBELET_EXTRA_ARGS=--fail-swap-on=false

        swapoff -a
vim /etc/fstab
# 注释掉swap的自动挂载

      2. 安装

        # 这个过程需要一些时间
kubeadm init --config=init-config.yaml
# 可能会遇到一些错误需要reset kubelet服务

# 修改权限
mkdir -p $HOME/.kube
sudo cp -i /etc/kubernetes/admin.conf $HOME/.kube/config
sudo chown $(id -u):$(id -g) $HOME/.kube/config

        注意记录安装成功后的信息 kubeadm join xxxxxxxx

    7. Node节点
      前面的准备工作都是一样的 只需要在最后生成配置文件后执行
      kubeadm join --config=join-config.yaml

    8. 如果你只想安装单节点的话

      kubectl taint nodes --all node-role.kubernetes.io/master-
