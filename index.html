hexo添加评论系统

发表于 2020-10-21 分类于 hexo
  1. hexo支持很多评论系统，由于我的blog是部署在github page 上，所以，索性我就直接用gitalk用做评论系统
  2. 配置方法
    1. 首先先在github中创建一个用于存储评论的公有仓库，注意（博客可以部署在私有仓库，评论必须public）
    2. 注册Github APP https://github.com/settings/applications/new
      • Homepage URL 与 Authorization callback URL 都填写你的blog地址
      • 其余随意填写
      1. 注册完毕我们会得到 ClientID 和Client Secret
      2. 在next主题配置文件中 找到如图的配置项
        gitalk:
  enable: true # 开启
  github_id: 你的账户
  repo: 刚刚创建的评论仓库
  client_id: ClientID
  client_secret: Client Secret
  admin_user: 你的账户
      3. 配置好后重新部署即可
