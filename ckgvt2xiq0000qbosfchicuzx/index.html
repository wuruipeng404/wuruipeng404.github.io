数据结构之链表

发表于 2020-10-30 更新于 2020-11-02 分类于 数据结构 阅读次数：

1. 什么是链表

链表是由一组不必相连（可以连续也可以不连续）的内存结构节点，按特定的顺序链接在一起的抽象数据类型

2. 链表的分类

链表常用的有三类，单链表 双向链表 循环链表, 循环链表又分为单项与双向

3. 用Python来实现单向循环链表,其他的大同小异

一般会抽象如下方法

  • add(item) 向链表添加数据项
  • remove(item) 删除链表中的数据项
  • exist(item) 数据是否存在于链表
  • empty() 链表是否为空
  • size() 返回链表数据个数
# 链表的一个节点
class Node(object):
    def __init__(self, data=None):
        self.__data = data
        self.__next = None

    def get_data(self):
        return self.__data

    def set_data(self, data):
        self.__data = data

    def get_next(self):
        return self.__next

    def set_next(self, node):
        self.__next = node


# 单向循环链表
class LinkList(object):
    def __init__(self):
        self.head: Node = Node()
        # 初始化时,下一个节点的指针指向head
        self.head.set_next(self.head)

    # 添加节点
    def add(self, item):
        node = Node(item)
        # 将新节点的指针指向 head
        node.set_next(self.head.get_next())
        # 将head指针指向 新创建的node
        self.head.set_next(node)

    # 移除节点
    def remove(self, item):
        prev = self.head
        while prev.get_next() != self.head:
            cur = prev.get_next()

            if cur.get_data() == item:
                prev.set_next(cur.get_next())

            prev = prev.get_next()

    def exist(self, item) -> bool:
        cur = self.head.get_next()
        while cur != self.head:
            if cur.get_data() == item:
                return True
            cur = cur.get_next()
        return False

    def empty(self):
        return self.head.get_next() == self.head

    def size(self):
        count = 0
        cur = self.head.get_next()
        while cur != self.head:
            count += 1
            cur = cur.get_next()
        return count


if __name__ == "__main__":
    s = LinkList()
    print(f"s.empty() == {s.empty()}, s.size() == {s.size()}")

    s.add(100)
    s.add(99)
    print(f"s.empty() == {s.empty()}, s.size() == {s.size()}")

    print(f"100 {'' if s.exist(100) else 'not'} in s")
    print(f"98 {'' if s.exist(98) else 'not'} in s")
    print(f"s.empty() == {s.empty()}, s.size() == {s.size()}")

    s.remove(99)
    print(f"s.empty() == {s.empty()}, s.size() == {s.size()}")
